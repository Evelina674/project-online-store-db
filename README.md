# База даних онлайн-магазину одягу

Цей репозиторій містить проєкт курсової роботи з дисципліни **«Бази даних»**.

## Структура репозиторію

- `docs/vision.md` – Опис бачення (Vision Document)
- `docs/brd.md` – Бізнес-вимоги та функціональні вимоги (BRD)
- `docs/nfr.md` – Нефункціональні вимоги (NFR)
- `docs/schema.md` – Опис схеми бази даних
- `docs/instructions.md` – Інструкція з розгортання
- `db/ddl.sql` – SQL-скрипт створення структури бази даних
- `db/data.sql` – SQL-скрипт наповнення тестовими даними

## Як запустити

1. Створити базу даних `clothing_store` у MySQL.
2. Виконати скрипт `db/ddl.sql` (створення таблиць).
3. Виконати скрипт `db/data.sql` (додавання тестових даних).
